# Expense API

A training REST API built with **Java + Spring Boot** for managing personal expenses.

The goal of this project is to practice backend development concepts such as CRUD operations, DTOs, pagination, filtering, sorting, validation, global exception handling, logging and API documentation with Swagger/OpenAPI.

## Tech Stack

- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- Bean Validation
- Lombok
- Swagger / OpenAPI
- Maven

## Features

- Create expenses
- List expenses with pagination
- Filter expenses by category
- Get expense by ID
- Update expense
- Delete expense
- Sort expenses by creation date
- Filter expenses by minimum and maximum amount
- Filter expenses by date range
- Search expenses by partial title
- Global exception handling
- Swagger API documentation
